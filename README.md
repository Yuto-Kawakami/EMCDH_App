## run web broser 
```
$ ionic serve
$ ionic cordova run browser
```

## run ios emulator
``` 
$ ionic cordova build ios
$ ionic cordova emulate ios --target iPhone-7 --livereload
```

## build ios production 
```
$ ionic cordova build ios --prod
open /platforms/ios/赤ちゃんとママを守る防災ノート.xcodeproj in Xcode
build as usual
```
